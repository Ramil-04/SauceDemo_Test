SauceDemo Test Automation

Bu layihə SauceDemo veb tətbiqi üçün avtomatlaşdırılmış UI testlərini ehtiva edir. Testlər Java və Selenium WebDriver istifadə edilərək hazırlanmışdır.

Texnologiyalar
Java
Selenium WebDriver
TestNG
Maven
Layihə Quruluşu
src/
├── main/
└── test/
Tələblər
Java 8 və ya daha yeni versiya
Maven
Chrome Browser
Quraşdırma

Repository-ni klonlayın:

git clone <repository-url>

Layihə qovluğuna keçin:

cd SauceDemo_Test

Asılılıqları yükləyin:

mvn clean install
Testlərin İşə Salınması

Bütün testləri işə salmaq üçün:

mvn test
